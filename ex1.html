USNA STEM Logic Module - Progamming problems

Your job for this activity is to write code that implements the same functionality as a not-gate, an and-gate, and an or-gate. Your tools are the simple programming language features we've discussed so far: if/if-else-statements, testing for equality with the == operator, and a simple "API" of functions getA(), getB() and output(num), which we've added to allow your program to communicate with the environment in which the program operates. If you have some programming experience, please don't cheat and use language features other than the ones listed above! The goal is to do do this only with if/if-else and ==.
Your API: An "API" is a set of functions your program can use to interact with the environment it operates in. Your API for this problem consists of three functions:
  1. getA() - gets the value of the "A" input to the program. E.g. "if (getA() == 0) ...".
  2. getB() - gets the value of the "B" input to the program. E.g. "if (getB() == 0) ...".
  3. output(num) - outputs the value num from the program. E.g. "output(1);"
Note: The code editor starts with a simple program that just outputs whatever value input A has, ignoring input B alltogether.
